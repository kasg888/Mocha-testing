# Mocha-testing
Learning to do automation test using mocha

To run this repo
1. Clone this repository
```
git clone https://github.com/alvintrianto/myskill-api-test.git
```
2. Open the folder with VS Code
3. Install npm with
```
npm install
```
4. Run mocha test 
```
npm run test {path/to/file}.js
```
